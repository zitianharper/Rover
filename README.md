Rover rover rover, I'm coming over over over 

What this project does:
- A C# application that stimulates a Rover that moves around with user inputs. This is an independent project with following requirements:
	- 2 days time limit
	- TDD required
	- Handling invalid inputs

Features
- Grid-based navigation with X Y coordinates
- Direction handling - North, East, South, West
- Command processing
	- L - Turn 270%
	- R - Turn 90%
	- M - Move forward by 1 grid point
